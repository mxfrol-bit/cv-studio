AI-скрининг кандидатов — NeuroStaff

500+ резюме в пакете, AI-голосовое интервью, автоматический топ-5 для рекрутера. Скорость найма ×5.

Задача клиента

Рекрутинговое агентство: 500+ откликов в неделю на однотипные позиции. Рекрутеры тратили 80% времени на первичный отсев, не успевая работать с перспективными.

Что сделали

  • Подключение к hh.ru API для автоматического получения откликов
  • NLP-анализ резюме: соответствие требованиям, красные флаги, релевантный опыт
  • Голосовое AI-интервью: 15 вопросов, анализ ответов и тональности
  • Автоматический рейтинг топ-5 кандидатов с обоснованием
  • Рекрутер видит готовый список с записями и резюме

Технологический стек

  • hh.ru API — получение откликов
  • spaCy + GPT-4o — NLP-анализ резюме
  • Retell AI — голосовые интервью
  • Supabase — хранение данных и рейтингов

Результат

Скорость найма ×5. Рекрутеры работают только с топ-5 кандидатами. Стоимость закрытия одной вакансии снизилась на 70%. Платформа стала SaaS-продуктом для 3 других агентств.
