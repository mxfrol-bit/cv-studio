Персонализированная воронка продаж

ML-рекомендации в email-канале на основе истории 1.2M клиентов. Автоматическая RFM-сегментация и персонализация в реальном времени.

Задача клиента

FMCG-сеть (87 магазинов, 1.2M держателей карты лояльности). Базовые массовые рассылки давали конверсию 1.2%. Задача: выйти на 1.5% без роста частоты отправок.

Что сделали

  • Интеграция с CRM и транзакционной базой (PostgreSQL)
  • ML-модель предсказания следующей покупки на основе RFM + поведенческих факторов
  • Динамическая персонализация: товар, цена, время отправки — индивидуально
  • A/B тестирование и мониторинг в реальном времени
  • Обучение маркетинговой команды работе с дашбордом

Технологический стек

  • Python + scikit-learn — ML-рекомендации
  • Supabase — хранение векторных эмбеддингов
  • N8N — автоматизация отправок и триггеров
  • FastAPI — API для интеграции с ESP

Результат

Конверсия выросла с 1.2% до 1.5% (+25%). При базе 1.2M — это +3 600 продаж на каждую рассылку. Средний чек персонализированных писем на 18% выше. ROI достигнут за 3 месяца.
