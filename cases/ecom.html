Динамическое ценообразование на маркетплейсах

ML-модель мониторит 1500+ конкурентов и спрос. Автоматическая переоценка 10 000+ SKU в реальном времени.

Задача клиента

Крупный продавец на Wildberries (10 000+ SKU). Ручное управление ценами занимало 3 FTE и давало запаздывание 24-48 ч. Конкуренты реагировали быстрее — продавец терял долю.

Что сделали

  • Парсинг цен 1500+ конкурентов каждые 2 часа
  • ML-модель на основе спроса, остатков, сезонности и конкурентов
  • Автоматическое обновление цен через WB API
  • Дашборд с аналитикой по категориям и SKU
  • A/B тесты ценовых стратегий

Технологический стек

  • Python + LightGBM — ML-модель ценообразования
  • WB API — обновление цен
  • Supabase — хранение истории цен
  • Grafana — аналитический дашборд

Результат

Маржинальность выросла на 20%. Время реакции на изменения рынка: с 48 ч до 2 ч. 3 FTE переведены на стратегические задачи. ROI — 2.5 месяца.
