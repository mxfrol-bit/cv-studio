AI-скоринг кредитных заявок

LLM + ML анализирует нестандартные данные заёмщика. NPL снизился на 30%. Скорость принятия решений: с 4 ч до 12 мин.

Задача клиента

МФО с объёмом выдач 200+ заявок в день. Классический скоринг пропускал 30% "серых" заёмщиков, по которым не было стандартных данных. NPL рос. Нужен был нестандартный подход.

Что сделали

  • LLM-анализ текстовых данных заявки: манера изложения, логика описания занятости
  • Поведенческий скоринг: время заполнения, паузы, исправления в анкете
  • Интеграция с БКИ и открытыми реестрами
  • Human-in-the-loop для пограничных случаев
  • Система объяснимости решений (XAI) для регулятора

Технологический стек

  • XGBoost + GPT-4o — гибридный скоринг
  • FastAPI — API для кредитной системы
  • Supabase — хранение векторов заявок
  • Grafana — мониторинг качества модели

Результат

NPL снизился с 8% до 5.6% (−30%). Скорость принятия решения: с 4 часов до 12 минут. Конверсия в выдачу выросла на 12% за счёт снижения ложных отказов.
